Создаем статичный веб-сайт с помощью Amazon S3

В этой лабораторной работе мы с вами создадим простой статичный веб-сайт и зададим настройки его конфигурации. Этот пример продемонстрирует как создать экономически эффективный хостинг для веб-сайта, который будет содержать файлы: HTML, CSS, JavaScript, шрифты и изображения.

Функции навигации по сайту и функция поиска не работают, т.к. цель данныой лабораторной работы отработкка навыков размещения файлов сайта и его запуска на Amazon S3.
